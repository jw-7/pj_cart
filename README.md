# 1. 프로젝트 개요

* 'pj_cart'는 프로젝트 카트의 줄임말로 목적있는 쇼핑을 하기 위한 쇼핑몰 사이트 입니다.  

* 프로젝트를 구현함으로써 자주 사용했던 기술들의 향상과  
새로운 기술 및 개념들을 파악하기 위해서 이 프로젝트를 시작했습니다.   

# 2. 프로잭트 개발환경 및 기술스텍

* Language  
\- Java(JDK1.8,JSP), HTML/CSS, JavaScript

* Framework  
\- Spring, Mybatis, Spring Security, BootStrap

* Server  
\- Apache Tomcat v8.5

* DataBase  
\- PostgreSQL 42.4.1

* API & Library  
\- 다음 우편번호 검색 API, Google Mail SMTP, 아임포트 결제 API, Google 로그인 API, jQuery

* Tool  
\- Spring Tool Suite 4, DBeaver 21.1.4 , Maven, Github

* 운영체제  
\- Windows 11

# 3. 프로젝트 구현기능  

* 회원(User)
  * 로그인 / 로그아웃, Google 아이디 로그인
  * 회원가입, 회원정보 보기, 회원정보 수정, 회원탈퇴
  * 아이디 / 비밀번호 찾기
* 상품(Goods)
  * 상품등록 , 상품조회, 상품수정, 상품삭제
  * 상품정렬 , 상품검색, 상품 상세조회,
  * 상품후기 등록 / 수정 / 삭제, 상품후기 댓글등록 / 수정 / 삭제  
* 주문(Order)
  * 상품주문, 상품 주문내역 조회, 상품 주문취소
* 장바구니(Cart)
  * 장바구니 담기, 장바구니 수정, 장바구니 삭제, 장바구니 조회, 장바구니 상품주문
* 결제(Payment)
  * 카카오페이 결제
* 관리자 게시판(AddrBoard)
  * 공지사항 게시판 조회 / 등록 / 수정 / 삭제
  * 1:1 문의 메일보내기
  * 상품 / 주문 / 회원 관리
