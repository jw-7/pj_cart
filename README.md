# 1. 프로젝트 개요

* 'pj_cart'는 프로젝트 카트의 줄임말로 목적있는 쇼핑을 하기 위한 쇼핑몰 사이트 입니다.  

* 프로젝트를 구현함으로써 자주 사용했던 기술들의 향상과  
새로운 기술 및 개념들을 파악하기 위해서 이 프로젝트를 시작했습니다.   

# 2. 프로잭트 개발환경 및 기술스텍

* Language 
\- Java(JDK1.8),HTML/CSS, JavaScript

* Framework  
\- Spring, Mybatis, Spring Security, BootStrap

* Server  
\- Apache Tomcat v8.5

* DataBase  
\- PostgreSQL 42.4.1

* API & Library  
\- KAKAO JQuery

* Tool  
\- Spring Tool Suite 4, DBeaver 21.1.4 , Maven
